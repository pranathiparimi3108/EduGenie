# User Guide

## Getting started

1. Open EduGenie in your browser after starting the backend.
2. Enter a topic (for most features).
3. Select a feature from the dropdown.
4. Click **Generate**.
5. Read the AI-generated response in the result panel.

## Feature guide

### Explain Topic
Best for learning a new concept from scratch.

### Learning Path
Creates a multi-day study plan for structured learning.

### Generate Quiz
Use after studying a topic to test your understanding.

### Ask a Question
Ask specific doubts with optional subject context.

### Summarize Content
Paste long notes to get a quick revision summary.

## Tips

- Use specific topics for better results
- Keep questions clear and focused
- Add context in Q&A for more accurate answers
